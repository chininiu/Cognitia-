<h1>COGNITIA</h1>

<h3> What is Cognitia? </h3>
<div> A game that generates its own maze with the use of DFS algorithm. It integrates active recall learning strategy and uses user made flashcards that are scattered and
  can be access/answered within the maze.</div>
<hr>
<h3>Purpose</h3>
<div>To analyze the impact of maze system towards the active recall strategy, if it will enhance the exisitng strategy or not</div>
<hr>
<h3>TOOLS</h3>
<div>Made with godot and its built in language, GDSCRIPT.</div>
<hr>
<h3>How to play</h3>
<div>Make sure you have the Godot engine installed in your device. You can clone this repository and run with the Godot Engine</div>
